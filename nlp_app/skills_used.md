# Skills Used in TextChar AI Project

## Core Programming Languages
- **Python** - Primary language for entire application
- **JavaScript** - Frontend interactivity
- **HTML/CSS** - UI structure and styling

## Web Development
- **Flask** - Web framework for routing and API endpoints
- **Jinja2** - Template rendering engine
- **REST API** - RESTful endpoints under `/api`

## Machine Learning & AI
- **PyTorch** - Deep learning framework for GPU acceleration
- **Transformers** - Hugging Face library for NLP models
- **TensorFlow** - Machine learning framework (firsthand experience)
- **Scikit-learn** - ML algorithms and model evaluation

## NLP Libraries & Models
- **KeyBERT** - Keyword extraction with sentence transformers
- **spaCy** - Named Entity Recognition (en_core_web_md)
- **DistilBERT** - Sentiment analysis (SST-2 fine-tuned)
- **T5-small** - Text summarization (60MB model)
- **DistilBART** - Alternative summarization model
- **BART-large-CNN** - Fallback summarization

## Browser Automation
- **Playwright** - Headless browser for URL content extraction
- **BeautifulSoup4** - HTML parsing and text extraction
- **requests** - HTTP requests for web content

## Data Processing
- **NumPy** - Numerical computations
- **Pandas** - Data manipulation and analysis

## Computer Vision
- **OpenCV** - Video processing and computer vision techniques
- **MoviePy** - Video editing automation
- **Vosk** - Subtitle generation

## Database Technologies
- **MySQL** - Primary database with connector-python
- **SQLite** - Fallback database for local development
- **SQL** - Queries, joins, subqueries, window functions

## Security & Authentication
- **bcrypt** - Password hashing for user authentication
- **Session Management** - Flask session handling

## Development Tools
- **VS Code** - Primary IDE
- **Jupyter Notebook** - Prototyping and experimentation
- **Git/GitHub** - Version control and collaboration
- **Virtual Environment (venv)** - Dependency isolation

## Data Science Fundamentals
- **EDA** - Exploratory Data Analysis
- **Data Cleaning** - Preprocessing and transformation
- **Model Training** - ML model development
- **Model Evaluation** - Performance assessment

## Excel & Spreadsheets
- **Pivot Tables** - Data summarization
- **VLOOKUP/XLOOKUP** - Data lookup functions
- **Conditional Formatting** - Visual data highlighting
- **Statistical Analysis** - Basic statistical functions

## Hardware & Performance
- **GPU Acceleration** - CUDA support with RTX 3050 (6GB VRAM)
- **Model Optimization** - Quantization and efficient inference

## UI/UX Features Implemented
- Live character/word count
- Analysis history tracking
- Tab-based navigation
- Toast notifications
- Responsive design

## Additional Skills
- **Algorithm Design** - Text classification without ML (rule-based)
- **ETL Pipelines** - Data extraction, transformation, loading
- **API Integration** - Internal and external API consumption
- **Error Handling** - Comprehensive try-except with user-friendly messages
